# App Brewery Course Projects

This repository contains projects I built as part of Angela Yu's course, [The Complete Full-Stack Web Development Bootcamp](https://www.udemy.com/course/the-complete-web-development-bootcamp/).

Each project helped me practice a different part of the full-stack web development workflow, from static front-end structure to server-side routing and data persistence.

## What I Learned In The Course

Through this bootcamp, I built a stronger foundation in both front-end and back-end web development by working through real projects and hands-on exercises. The course helped me understand how the internet works, how websites are structured, and how full-stack applications are planned, built, and improved using professional development workflows.

Some of the main skills and topics I learned throughout the course include:

- HTML5, CSS3, Flexbox, Grid, and Bootstrap 5 for building responsive and well-structured user interfaces.
- JavaScript ES6, DOM manipulation, and jQuery for adding interactivity and dynamic behavior to web pages.
- Node.js, npm, Express.js, and EJS for creating server-side applications and rendered web experiences.
- REST concepts, APIs, SQL, and PostgreSQL for building data-driven applications and working with persistent storage.
- Git, GitHub, Bash command line tools, and general developer best practices for managing projects and working more professionally.
- React and React Hooks for building modern front-end applications with reusable components.
- Authentication, deployment workflows, and broader full-stack architecture concepts that connect front-end and back-end systems.
- Exposure to newer topics like Web3, blockchain concepts, token contracts, NFTs, and decentralized app development.

The course also emphasized learning by building, which helped me improve problem-solving, project structure, debugging, and confidence in creating complete web applications from scratch.

## Projects

### About Me (Capstone Project)

This project helped me learn how to build and structure a personal portfolio-style page using HTML, CSS, and Bootstrap. It gave me practice with responsive layout design, typography, spacing, image presentation, and organizing sections like a hero area, introduction, contact call-to-action, and footer.

### Book Notes (Capstone Project)

This project helped me learn how to build a full-stack CRUD application with Express, EJS, and PostgreSQL. It gave me hands-on experience with routing, form handling, server-side rendering, database connections, SQL queries, editing and deleting records, and structuring an application that stores and displays user-created content.

### Simple Post (Capstone Project)

This project helped me learn how to build a simple blog-style app with Express and EJS while working with file-based data storage. It gave me practice with handling form submissions, creating, editing, and deleting posts, reading and writing JSON data, validating user input, and organizing routes for a small content management workflow.
