# App Brewery Course Projects

This repository contains projects I built as part of Angela Yu's course, [The Complete Full-Stack Web Development Bootcamp](https://www.udemy.com/course/the-complete-web-development-bootcamp/).

Each project helped me practice a different part of the full-stack web development workflow, from static front-end structure to server-side routing and data persistence.

## Projects

### About Me (Capstone Project)

This project helped me learn how to build and structure a personal portfolio-style page using HTML, CSS, and Bootstrap. It gave me practice with responsive layout design, typography, spacing, image presentation, and organizing sections like a hero area, introduction, contact call-to-action, and footer.

### Book Notes (Capstone Project)

This project helped me learn how to build a full-stack CRUD application with Express, EJS, and PostgreSQL. It gave me hands-on experience with routing, form handling, server-side rendering, database connections, SQL queries, editing and deleting records, and structuring an application that stores and displays user-created content.

### Simple Post (Capstone Project)

This project helped me learn how to build a simple blog-style app with Express and EJS while working with file-based data storage. It gave me practice with handling form submissions, creating, editing, and deleting posts, reading and writing JSON data, validating user input, and organizing routes for a small content management workflow.
